Properties
Name Type Description
lists Array.<Array.<string>> 将要创建的对应文件的 title 和 path 组合数组，默认继承自属性：`this._conf_lists`。
isIndent boolean 是否缩进。如果` false `,则后续的与缩进相关的设置不会生效。默认：`true`。
TabLength number Tab键所占空格数，默认: `4`（这与你在编辑器上的设置没有关系，仅指单个缩进的所占字符长度）。

EncodeURI boolean 是否对 href 进行 encodeURI() 处理——带空格（汉字等）的路径会导致404。 - （当然，这取决于最终文档托管的平台如何处理，如果不确定保持默认值：` false `）。
indentByDir boolean 是否以文件夹作为缩进依据，默认：`false`。 - 如果` true`，那么将从当前目录的第二层文件夹开始缩进，默认地只会进行缩进最多2次而不管路径层级有多深。 - 如果` false `，则每一层文件夹都缩进——层级越多缩进长度越长。
