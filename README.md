# summarymd

#### 项目介绍

围绕 `SUMMARY.md` 文件工作的工具。

* 适用项目(平台)类型：
    - GitBook
    - 看云文档
* 适用场景：
    - 批量创建文档项目大纲中的文件。
    - 批量为已有文档创建目录列表文件。
    - 自动实时生成目录（*不会覆盖正式目录*）创建文件。
* 适合人群：
    - Markdown 重度用户。
    - 经常在文档、目录间切换的同学（我自己眼神不好，经常弄错章节顺序以及章节和文档路径的对应关系）。

#### 安装教程

1. 安装到项目：
   
   ```
   npm i summarymd
   ```

2. 全局安装：

    ```
    npm i -g summarymd
    ```
    > **推荐**： 以命令方式使用，最大限度降低对文档仓库的污染。
